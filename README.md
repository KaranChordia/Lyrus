# Lyrus Life Sciences Project

This repository serves as the central hub for our initiative to pitch and secure long-term work with Lyrus Life Sciences.

## Project Goals
1. **Deep Research & Documentation:** Understand their business, market position, and operational needs.
2. **Page/Web Development:** Create compelling web pages, presentations, and landing pages tailored for them.
3. **AI Agents & Internal Tools:** Develop proofs-of-concept (POCs) for internal tools and AI agents to demonstrate our capabilities.
4. **Proposal Formulation:** Pitch for regular contracts in content creation, AI agentic solutions, and software development.

## Folder Structure
- `/research` - Research notes, findings, and documentation about Lyrus Life Sciences.
- `/proposal` - Drafts and final versions of the pitch proposal.
- `/pages` - Source code and mockups for web pages/content to show them.
- `/tools` - Code for POC internal tools and AI agents.
